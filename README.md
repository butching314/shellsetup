##HowTo

###Setup shortcuts
Update ~/.zshrc

Append
```
source /Usr/marvr/shellsetup/shellalias
```

###Setup GIT
Create your gitcongi link:
```
ln -s ~/shellsetup/gitconfig ~/.gitconfig
```

###Setup VIM
Install vim-plug:
https://github.com/junegunn/vim-plug

Create your vimrc link:
```
ln -s ~/shellsetup/vimrc ~/.vimrc
```

Load plugins.
